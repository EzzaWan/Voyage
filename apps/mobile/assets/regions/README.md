# Region Icons

Please download the following icons from Flaticon and place them in this folder:

1. **asia.png** - https://www.flaticon.com/free-icon/location_10601138
2. **europe.png** - https://www.flaticon.com/free-icon/gps_10601280
3. **north-america.png** - https://www.flaticon.com/free-icon/location_10601282
4. **south-america.png** - https://www.flaticon.com/free-icon/location_10601281
5. **africa.png** - https://www.flaticon.com/free-icon/location_10601276
6. **oceania.png** - https://www.flaticon.com/free-icon/map_10601284

**Instructions:**
1. Download each icon from the links above
2. Save them as PNG files with the exact names listed above
3. Place all 6 files in this `assets/regions/` folder
4. Recommended size: **256x256px** for best quality (they will be displayed at 32x32px in the app for crisp rendering on high-DPI displays)
